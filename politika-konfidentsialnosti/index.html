Политика конфиденциальности

Черновой шаблон — перед публикацией должен быть проверен и утверждён владельцем сайта (Оператором) либо юристом с указанием точных реквизитов.

Настоящая политика определяет порядок обработки персональных данных посетителей сайта cane-corso-pitomnik.ru (далее — «Сайт») питомника кане-корсо Tagma Optimatum.

1. Оператор персональных данных

Оператором обработки персональных данных является [ФАМИЛИЯ, ИМЯ, ОТЧЕСТВО ЗАВОДЧИКА], далее — «Оператор». Контакты Оператора указаны на странице «Контакты».

2. Какие данные собираются

При заполнении форм на Сайте (форма обратной связи) может собираться: имя, номер телефона, содержание сообщения. Сайт не запрашивает специальные категории персональных данных.

3. Цели обработки

Персональные данные обрабатываются исключительно для связи с посетителем Сайта по вопросам, связанным с покупкой щенка, и не передаются третьим лицам, за исключением случаев, предусмотренных законодательством РФ.

4. Согласие на обработку

Отправляя форму на Сайте, пользователь подтверждает согласие на обработку указанных персональных данных в целях, описанных в разделе 3.

5. Хранение и защита данных

Оператор принимает необходимые организационные и технические меры для защиты персональных данных от неправомерного доступа, изменения, раскрытия или уничтожения.

6. Права пользователя

Пользователь вправе запросить у Оператора информацию о своих персональных данных, их уточнение или удаление, обратившись по контактам, указанным на странице «Контакты».

7. Файлы cookie и аналитика

Сайт может использовать счётчики аналитики (Яндекс.Метрика, Google Analytics) для оценки посещаемости. На момент публикации данного черновика идентификаторы счётчиков не подключены — при их подключении данный раздел должен быть дополнен точным описанием используемых сервисов.

8. Изменения политики

Оператор вправе вносить изменения в настоящую политику. Актуальная версия всегда доступна по этому адресу.
